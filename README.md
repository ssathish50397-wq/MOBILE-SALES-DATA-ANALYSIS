## **📊 Mobile Sales Data Analysis**

**📌 Project Overview**

This project focuses on analyzing mobile sales data to understand brand performance, revenue trends, and customer behavior. The analysis helps businesses make data-driven decisions in a competitive smartphone market.

**📁 Dataset Information**

Source: Kaggle
Format: CSV
Rows: 500
Columns: 13

🔑 Features Included:

Customer Details: Customer_ID, Country

Product Details: Brand, Model, Storage, Color, Price_USD

Sales Details: Revenue_USD, Sale_Month, Sale_Year

Payment Info: Payment_Method

⚙️ Tools & Technologies

Python 🐍
Pandas
Matplotlib
Excel

**🔄 Workflow**

**1. Data Collection**

Dataset downloaded from Kaggle
Checked rows, columns, and structure

**2. Data Understanding**

Identified business problem
Explored features and relationships

**3. Data Cleaning**

Checked missing values using isnull()
Verified duplicates
Dataset found clean

**4. Data Transformation**

Prepared data for analysis
Structured features for easy comparison

**5. Data Visualization**

Created charts to analyze trends and insights

**📈 Key Analysis & Insights**

**💰 Revenue Analysis**

Total Revenue: 4,639,641
Average Revenue: 9,279
Maximum Revenue: 29,660

**💳 Payment Method Analysis**

Cash and Cards dominate transactions
Online & installment methods are less used

**📱 Brand Performance**

Vivo → Highest sales volume
Apple → Highest revenue
OnePlus → Lowest sales

**🌍 Country-wise Sales**

Top Markets: Australia, USA, India
Lowest Contribution: Canada, Saudi Arabia

**📊 Monthly Sales Trend**

Peak Sales: Month 6
Lowest Sales: Month 5
Sales show seasonal variation

**💾 Storage Preference**

Most Preferred: 256GB
Least Preferred: 64GB
Customers prefer mid-high storage options

**🔍 Final Insights**

Apple leads in both sales and revenue
Market is highly competitive with similar brand performance
Traditional payment methods still dominate
Customers prefer high storage & premium devices
Seasonal trends affect sales performance

**🚀 Conclusion**

This project demonstrates how data analysis using Pandas and visualization tools can uncover meaningful business insights and improve decision-making in the mobile industry.
